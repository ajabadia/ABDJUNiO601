OK! This is from the Juno 106 service manual:

To enter the test mode, hold KEY TRANSPOSE down and turn the JUNO-106 ON;
the display window will read :

        ----  ----
        |  |  |  |
        ####  ----    (#### = Segment on)
        |  |  |  |
        ----  ####

        indicating that the unit is in the test mode. During the test mode,
each switch serves as follows :

==============================================================================
| SWITCH              |     FUNCTION DURING TEST MODE                        |
==============================================================================
| KEY        | POLY 1 | UNISON:  All six modules are assigned simultaneously |
| ASSIGNMENT |        |          to a key being pressed.                     |
|            |--------|------------------------------------------------------|
|            | POLY 2 | NON      The voices are assigned to the keys played  |
|            |        | ROTARY:  in the order of CH1 to CH6 as long as the   |
|            |        |          previous keys are held down.                |
|            |        |          One-key staccato always sounds CH1 only.    |
|            |        |                                                      |
|            |        | The display window indicates current channel number. |
|            |--------|------------------------------------------------------|
|            | POLY 1 | ROTARY:  The voices are assigned in a cyclic manner; |
|            |   &    |          7th key steals the voice from the 1st key.  |
|            | POLY 2 |                                                      |
|            |        |                                                      |
|            |        | The display window indicates current channel number. |
==============================================================================
| BANK       | GROUP A| HOLD OFF                                             |
| GROUP      | GROUP B| HOLD ON                                              |
==============================================================================
| TAPE       | SAVE   | MIDI FUNCTION II CHECK                               |
| CHECK LED  | VERIFY | MIDI FUNCTION I CHECK                                |
==============================================================================
| MIDI CH    |        | Turns D/A output to 0V                               |
==============================================================================


Pressing BANK buttons also evokes Test Program and sets the front panel 
controls as below. PATCH buttons have no effect in the test mode.

Edit functions are also active in test mode; when an edit is made, display 
window lights a dot. To return to the test mode, press the same BANK button
again.
